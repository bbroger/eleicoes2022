Políticas de Privacidade

Ao acessar este site você está de acordo com estes termos.

Considerações iniciais:

  • Este site não tem nenhuma ligação com a Justiça Eleitoral, com o Tribunal Superior Eleitoral ou qualquer orgão regulamentador das eleições brasileiras
  • Os votos exibidos neste site não irão de maneira nenhuma interferir no processo eleitoral de 2022
  • Este site não tem nenhuma ligação com orgãos de pesquisas, como por exemplo o IBOPE ou DataFolha

Privacidade de seus dados:

Para garantir a idoneidade dos resultados, solicitamos ao visitante que cadastre-se neste site, informando Nome, Sobrenome, CPF, Data de Nascimento, Login e Senha, como também responder ao sistema reCaptcha.

Os dados informados passam por um avançado processo de criptografia e são salvos em um banco de dados devidamente criptografados, de forma que não possam ser visualizados por ninguém, nem mesmo os desenvolvedores deste site.

De forma nenhuma os dados inseridos neste serão publicados, vendidos, utilizados para entrar em contato ou de qualquer outra forma, que não seja para logar-se e votar.

Processo de votação:

O processo de votação foi desenvolvido de forma que cada usuário possa votar somente uma vez, de forma definitiva.

Após o usuário criar a conta e logar-se, será liberado a votação, onde o usuário deverá escolher, na página inicial, um candidato dos que estão disponíveis na lista, responder ao sistema reCaptcha e confirmar o voto.

Por segurança, após confirmado o voto, o usuário será deslogado do site. O usuário poderá logar-se novamente, mas não poderá votar outra vez.

O voto também é salvo no banco de dados de forma criptografada utilizando o mesmo sistema avançado de criptografia utilizado para criar a conta.

Cada candidato recebe um código numérico aleatório, e este código é renovado a cada cinco minutos, de forma que a URL de cada candidato não sempre a mesma.
